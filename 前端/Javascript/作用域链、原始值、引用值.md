##### 原始值与引用值、按值传递与按引用传递

`JavaScript`中的变量分为两种：

原始值（Boolean、String、Number、Symbol、Null、Undefined）

引用值（Object、Array）

保存原始值的变量按值访问，引用值变量按引用访问。

值传递：

```js
let a = 1;
let b = a;
// 按值传递，赋值变量a对应的值的副本给变量b

let foo = Object();
let bar = foo;
// 变量foo及bar都指向堆内存中同一个对象
// foo赋给bar的对象的引用而不是对象的副本
```
